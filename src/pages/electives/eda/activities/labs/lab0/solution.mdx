## Solution - NodeJS

Utilize https://github.com/ibm-messaging/event-streams-samples/blob/master/kafka-nodejs-console-sample/docs/Kubernetes_Service.md as needed for pointers.

1. Clone the git repository
    ```
    git clone https://github.com/ibm-messaging/event-streams-samples.git
    ```
1. Change directory
    ```
    cd event-streams-samples/kafka-nodejs-console-sample
    ```
1. Acquire the Service Credentials from the **Service Credentials** tab of your IBM Event Streams on Cloud instance and save it to a file `credentials.json`
1. Create kubernetes secret 
    ```
    kubectl create secret generic eventstreams-binding --from-file=binding=credentials.json
    ```
1. Deploy the App
    ```
    kubectl apply -f kafka-nodejs-console-sample.yaml
    ```
1. Wait and get the logs
    ```
    kubectl wait pod kafka-nodejs-console-sample --for=condition=Ready
    kubectl logs kafka-nodejs-console-sample --follow
    ```
1. Delete the pod
    ```
    kubectl delete pod kafka-nodejs-console-sample
    ```
1. Delete the topic if you are using the the free plan (free plan only allows one topic/partition)
